Linked List

Definition:

  • A linked list is a linear data structure where elements are not stored at contiguous memory locations.
  • Elements are linked using pointers. Each element, called a node, consists of two parts: data and a pointer (or link) to the next node in the sequence.
  • The first node is called the HEAD, and the last node's pointer is typically NULL, indicating the end of the list.

Working:

  • Insertion: A new node can be added at the beginning, end, or a specific position by adjusting the pointers of the adjacent nodes.
  • Deletion: A node is removed by modifying the pointer of the preceding node to point to the succeeding node, effectively bypassing the one being deleted.
  • Traversal: Starting from the HEAD, the list is traversed by following the `next` pointers until a NULL pointer is reached.

Time Complexity:

  • Access: O(n)
  • Search: O(n)
  • Insertion: O(1) (at head), O(n) (at tail/middle)
  • Deletion: O(1) (at head), O(n) (at tail/middle)

Space Complexity:

  • O(n) (to store n elements)
