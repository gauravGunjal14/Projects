Stack

Definition:

  • A stack is a linear data structure that follows the **Last-In, First-Out (LIFO)** principle.
  • The last element added to the stack is the first element to be removed.
  • Stacks are commonly used for tasks like undo/redo functionality, function call management, and expression evaluation.

Working:

  • The primary operations are **Push** (adding an element to the top) and **Pop** (removing the top element).
  • A stack pointer, often called `top`, keeps track of the most recently added element.
  • Other operations include **Peek** (viewing the top element without removing it) and **isEmpty** (checking if the stack is empty).

Time Complexity:

  • Push: O(1)
  • Pop: O(1)
  • Peek: O(1)
  • Search: O(n) (worst case)

Space Complexity:

  • O(n) (to store n elements)
